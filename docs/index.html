Weekly Brief – February 16, 2026

Cybersecurity Events & Incidents

  • LockBit resurgence targeting U.S. healthcare providers
    Multiple U.S. regional healthcare networks reported ransomware incidents attributed to LockBit affiliates. Initial access assessed via exposed RDP and credential stuffing. Encryption patterns and ransom notes aligned with LockBit 3.0 builds. Healthcare delivery systems experienced temporary service disruption. Source: https://www.bleepingcomputer.com/news/security/lockbit-ransomware-targets-us-healthcare-networks-in-new-wave/
  • Phishing campaign delivers DarkGate malware via Microsoft Teams
    Researchers identified a social engineering campaign leveraging Microsoft Teams external messaging to deliver DarkGate loaders. TTPs include signed MSI payloads and AutoIT obfuscation. Financial and manufacturing sectors primarily impacted. Source: https://www.securityweek.com/darkgate-malware-spread-via-microsoft-teams-phishing-campaign/
  • Volt Typhoon activity observed in U.S. critical infrastructure
    Ongoing living off the land techniques detected in energy and water utilities. Activity consistent with prior PRC aligned Volt Typhoon tradecraft, including use of valid credentials and network edge device exploitation. Source: https://www.cisa.gov/news-events/alerts/2026/02/12/prc-linked-volt-typhoon-activity-critical-infrastructure

Vendor & Security Product Updates

  • Microsoft February Patch Tuesday
    Microsoft addressed multiple critical vulnerabilities including remote code execution flaws affecting Windows Server and Exchange. Several CVEs added to CISA KEV catalog. Source: https://msrc.microsoft.com/update-guide/releaseNote/2026-Feb
  • Fortinet releases fixes for FortiOS SSL VPN flaw
    Fortinet patched a critical authentication bypass vulnerability affecting FortiOS SSL VPN configurations. Exploitation attempts observed in the wild. Source: https://www.fortiguard.com/psirt/FG-IR-26-045

Geopolitical & Political Developments

  • U.S. sanctions cyber operators tied to Iranian APT activity
    The U.S. Treasury sanctioned individuals associated with cyber operations targeting defense contractors and regional government entities. Source: https://home.treasury.gov/news/press-releases/jy2105

Notable Vulnerabilities & Patches

  • CVE-2026-21415, Windows RCE vulnerability
    Critical RCE vulnerability affecting Windows LDAP services. Exploitable via specially crafted network packets without authentication. Source: https://nvd.nist.gov/vuln/detail/CVE-2026-21415
  • CVE-2026-30102, Cisco IOS XE privilege escalation
    Improper input validation allows authenticated attackers to escalate privileges to root. Source: https://tools.cisco.com/security/center/content/CiscoSecurityAdvisory/cisco-sa-iosxe-pe-2026
