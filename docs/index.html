Weekly Brief - March 11, 2026

Cybersecurity Events & Incidents

  • Oracle E-Business Suite campaign leads to confirmed breach
    Michelin confirmed a data breach related to a large scale campaign exploiting vulnerabilities in Oracle E-Business Suite. The operation has been attributed to the Cl0p ransomware and extortion group. Attackers reportedly exploited previously unknown flaws to extract enterprise data from affected organizations.
    Source: https://www.securityweek.com/michelin-confirms-data-breach-linked-to-oracle-ebs-attack/
  • Salesforce ecosystem targeted in mass credential theft campaign
    The ShinyHunters cybercrime group reportedly launched a new campaign targeting hundreds of Salesforce customers. The operation focuses on credential harvesting and data exfiltration against enterprise cloud tenants, potentially impacting multiple SaaS environments.
    Source: https://www.securityweek.com/hundreds-of-salesforce-customers-allegedly-targeted-in-new-data-theft-campaign/
  • npm supply chain attack distributes PhantomRaven malware
    A renewed supply chain attack targeting the npm ecosystem is distributing the PhantomRaven malware. Malicious packages attempt to steal developer secrets and environment tokens. The campaign demonstrates continued targeting of CI/CD pipelines and developer environments.
    Source: https://cyberpress.org/phantomraven-targets-npm-supply-chain/
  • AI tool cloning campaign delivers malware through InstallFix
    Researchers identified a campaign distributing malware through cloned installation pages of AI developer tools. Malicious commands embedded in installation instructions execute payloads on victim systems, indicating increasing abuse of developer ecosystems for initial access.
    Source: https://www.securityweek.com/cloned-ai-tool-sites-distribute-malware-in-installfix-campaign/
  • Phishing campaigns deliver signed malware targeting Zoom and Teams users
    Multiple phishing campaigns are delivering signed malware disguised as workplace collaboration tools. The malware deploys Remote Monitoring and Management backdoors to maintain persistence and remote access.
    Source: https://cyberpress.org/signed-malware-targets-zoom-teams/
  • Energy sector incident claimed by ransomware group
    The Qilin ransomware group claimed responsibility for an attack on the Texas based electric cooperative Trinity Valley Electric Cooperative (TVEC), highlighting continued targeting of critical infrastructure organizations.
    Source: https://www.kaseya.com/blog/the-week-in-breach-news-03-11-26/

Vendor & Security Product Updates

  • Microsoft March 2026 Patch Tuesday
    Microsoft released updates addressing 79 vulnerabilities, including two publicly disclosed zero day vulnerabilities and multiple remote code execution flaws affecting enterprise software components.
    Source: https://www.bleepingcomputer.com/news/microsoft/microsoft-march-2026-patch-tuesday-fixes-2-zero-days-79-flaws/
  • Cisco patches critical firewall and management interface vulnerabilities
    Cisco issued patches addressing dozens of vulnerabilities in network security appliances. Two flaws affecting Secure Firewall Management Center received a maximum CVSS score of 10.0 and allow authentication bypass and remote code execution.
    Source: https://securityaffairs.com/188921/security/cisco-fixes-maximum-severity-secure-fmc-bugs-threatening-firewall-security.html
  • Multiple enterprise vendors issue coordinated vulnerability fixes
    Enterprise infrastructure vendors released patches addressing multiple high severity flaws including CVE-2026-23813, an authentication bypass vulnerability affecting network management interfaces with a CVSS score of 9.8.
    Source: https://thehackernews.com/2026/03/dozens-of-vendors-patch-security-flaws.html

Geopolitical & Political Developments

  • Iran linked MuddyWater targeting U.S. networks
    Iran associated threat actor MuddyWater deployed a new backdoor called Dindoor targeting U.S. organizations. The campaign reportedly focuses on intelligence gathering and network persistence through spear phishing and credential harvesting.
    Source: https://thehackernews.com/2026/03/iran-linked-muddywater-hackers-target.html
  • Russian cyber espionage targeting Ukrainian organizations
    Researchers observed phishing campaigns delivering two previously undocumented malware families targeting Ukrainian entities, indicating continued cyber activity aligned with the regional conflict environment.
    Source: https://therecord.media/russian-ukraine-hackers-malware
  • China linked cyber espionage campaign targeting Middle East organizations
    The Camaro Dragon group conducted espionage operations targeting organizations in Qatar. The campaign uses spear phishing and remote access malware to gain long term access to sensitive networks.
    Source: https://industrialcyber.co/ransomware/check-point-uncovers-china-linked-camaro-dragon-cyber-espionage-campaign-targeting-qatari-organizations/

  • CVE-2026-20079 and CVE-2026-20131, Critical vulnerabilities in Cisco Secure Firewall Management Center enabling authentication bypass and remote code execution with root privileges.

  • CVE-2026-26110 and CVE-2026-26113, Microsoft Office remote code execution vulnerabilities that could allow attackers to execute arbitrary code via malicious documents.
